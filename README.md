# Introduction
會有無人載具車的發想是緣於團隊成員在搬運重物時，覺得這個過程中太浪費勞力和時間，因此想到了，若能運用無人載具車來搬運，可望節省更多時間及人力成本，並且本團隊認為載具車的功能太過單一，所以附加了其他功能在載具車上，希望讓載具車的功能更加多元。
